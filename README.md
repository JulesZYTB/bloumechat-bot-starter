Welcome to the BloumeChat Example Bot repository. This project demonstrates how to use the official `bloumechat` SDK to build a functional bot.

The comprehensive documentation for both the SDK and this example bot is available in multiple languages inside the `docs/` folder.

<br/>

### Quick Setup

1. Copy `.env.sample` to `.env` and fill in your Token.
2. Run `npm install`
3. Run `npm run start` or `npx ts-node index.ts`

*For full details, please refer to the documentation links above!*
